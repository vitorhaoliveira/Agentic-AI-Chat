# @agentic-ai-chat/shared

Shared types, schemas, and utilities for the Agentic AI Chat system.

## Contents

- **types/**: TypeScript type definitions for chat, agents, and API
- **schemas/**: Zod validation schemas

## Usage

```typescript
import { Message, ChatRequest, loginSchema } from '@agentic-ai-chat/shared';
```

## Building

```bash
npm run build
```
